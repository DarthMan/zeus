# zeus
Next-generation AMXX C++ module featuring dynamic JIT trampolines, PIC-safe hooking, and native memory routing for GoldSrc.

Zeus is an advanced C++ module for AMX Mod X designed to push the limits of GoldSrc server modding. It features a custom JIT compiler, dynamic memory detouring with LDE trampolines, and automated PIC-safe hooking. By bridging PAWN scripts directly with raw engine memory, Zeus provides seamless support across standard HLDS, ReHLDS, and ReGameDLL environments without requiring hardcoded C++ forwards.
